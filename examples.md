# Порівняння растрової та векторної графіки

Щоб остаточно зрозуміти різницю, порівняємо ці два види графіки за основними технічними характеристиками.

### Зведена таблиця характеристик

| Характеристика | Растрова графіка | Векторна графіка |
| :--- | :--- | :--- |
| **Елемент побудови** | Піксель (точка) | Геометричний примітив (формула) |
| **Масштабування** | Якість втрачається (з'являється розмиття) | Якість ідеальна при будь-якому розмірі |
| **Фотореалістичність** | Дуже висока | Низька, зображення виглядають стилізовано |
| **Розмір файлу** | Залежить від роздільної здатності (зазвичай великий) | Залежить від складності об'єктів (зазвичай малий) |
| **Основні формати** | `.jpeg`, `.png`, `.gif`, `.webp`, `.tiff` | `.svg`, `.ai`, `.eps`, `.cdr` |
| **Програми для роботи** | Adobe Photoshop, GIMP, Photopea | Adobe Illustrator, Inkscape, CorelDRAW |

### Приклад з реального життя: Ефект наближення (Zoom)
Уявіть, що ви намалювали літеру "А". 
* Якщо це **растр**: при наближенні в 500% краї літери перетворяться на сходинки з сірих та чорних квадратів. 
* Якщо це **вектор**: при наближенні в 500% чи навіть 5000% край літери залишатиметься ідеально рівною та чіткою лінією.
