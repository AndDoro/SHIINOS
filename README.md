# Навчальні матеріали: Растрова та векторна графіка

Цей репозиторій містить повноцінний навчально-методичний комплекс з теми **«Растрова та векторна графіка»**, розроблений для учнів 10–11 класів (базовий рівень) відповідно до програми МОН України.

Матеріали створені з використанням технологій генеративного штучного інтелекту та оптимізовані для публікації через GitHub Pages.

## 📁 Структура проекту

* `index.html` — головна сторінка сайту з інтерактивним меню та навігацією.
* `theory/` — теоретичний блок (лекція, порівняльна таблиця, глосарій).
* `practice/` — практичний блок (аналітичні завдання, лабораторні роботи, творчий проект).
* `tests/` — інструменти контролю знань (експрес-тести, ситуаційні кейси).
* `resources/` — додаткові онлайн-інструменти та корисні посилання.
* `styles/custom.css` — кастомні стилі для адаптивного дизайну веб-сторінки.
