# Аналітичне завдання: Дослідження ефекту масштабування

### Мета:
Наочно побачити різницю між растровою та векторною графікою при зміні масштабу зображення.

### Хід роботи:
1. Завантажте на комп'ютер будь-яку фотографію у форматі `.jpg` або `.png`.
2. Відкрийте у браузері або завантажте будь-яку векторну іконку у форматі `.svg` (наприклад, з безкоштовного ресурсу Flaticon або подібного).
3. Відкрийте обидва файли у програмі для перегляду зображень або прямо у вкладках браузера.
4. Збільшіть масштаб фотографії (растру) до 500–1000%. Зверніть увагу на краї об'єктів. Чи з'явилися розмиті квадрати (пікселі)?
5. Збільшіть масштаб векторної іконки (вектору) до максимально можливого (500–1000%). Оцініть чіткість ліній та контурів.
6. Зробіть скріншоти ефекту наближення для обох зображень.

### Звіт:
Створіть короткий текстовий опис зафіксованих спостережень та додайте порівняльні скріншоти у свій звіт.
