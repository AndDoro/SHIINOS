# Додаткові ресурси для вивчення комп'ютерної графіки

Для глибшого занурення в тему та якісного виконання практичних завдань рекомендуємо скористатися наступними джерелами та інструментами.

---

## 🛠️ Безкоштовні онлайн-редактори

### Растрова графіка:
* [Photopea](https://www.photopea.com/) — потужний безкоштовний онлайн-редактор, який повністю повторює інтерфейс та функціонал Adobe Photoshop. Працює прямо у браузері.
* [GIMP](https://www.gimp.org/) — безкоштовна програма з відкритим кодом для завантаження на ПК (альтернатива Photoshop).

### Векторна графіка:
* [Vectr](https://vectr.com/) — простий та зрозумілий онлайн-редактор для створення векторних іконок, фігур та логотипів. Ідеальний для початківців.
* [Inkscape](https://inkscape.org/) — професійний безкоштовний векторний редактор з відкритим кодом для встановлення на комп'ютер.

---

## 📚 Навчальні матеріали та статті

* [Вступ до SVG від W3Schools](https://www.w3schools.com/html/html5_svg.asp) — інтерактивний туторіал, який показує, як влаштований код векторної графіки.
* [Шрифти та їхня природа](https://fonts.google.com/) — сервіс Google Fonts, де всі шрифти є чудовим прикладом використання векторної графіки на практиці.

---

## 🎨 Де шукати натхнення та безкоштовні заготовки?

* [Flaticon](https://www.flaticon.com/) — найбільша база безкоштовних векторних іконок у форматі `.svg` для ваших проектів.
* [Freepik](https://www.freepik.com/) — ресурс із мільйонами безкоштовних растрових фотографій та векторних ілюстрацій (звертайте увагу на формат завантаження).
* [Pixabay](https://pixabay.com/) — якісні фотографії та зображення без авторських прав для ваших лабораторних робіт.
