# Лабораторні роботи з комп'ютерної графіки

---

## Лабораторна робота №1: Базове редагування растрового зображення

**Платформа:** Онлайн-редактор [Photopea](https://www.photopea.com/) (безкоштовний аналог Photoshop) або програма GIMP.

### Завдання:
1. Відкрийте редактор Photopea у браузері.
2. Завантажте власне фото або будь-яке пейзажне зображення високої якості.
3. Виконайте кадрування (Crop) зображення, залишивши лише головний об'єкт по центру або за правилом третин.
4. Змініть параметри яскравості та контрастності: `Зображення` -> `Корекція` -> `Яскравість/Контраст`.
5. Застосуйте один із фільтрів для стилізації (наприклад, розмиття або підвищення різкості).
6. Збережіть підготовлений файл у двох форматах:
   * Звичайний `.jpeg` з якістю 80%.
   * Формат `.png` із підтримкою прозорості (попередньо вимкнувши видимість фонового шару).
7. Порівняйте розміри отриманих файлів у мегабайтах.

## Лабораторна робота №2: Створення векторного логотипу

**Платформа:** Онлайн-редактор [Vectr](https://vectr.com/) або програма Inkscape.

### Завдання:
1. Відкрийте векторний редактор.
2. Використовуючи інструменти геометричних примітивів (прямокутник, коло, еліпс), створіть основу для фірмового знака або логотипу (наприклад, значок хмаринки, шестерні або щита).
3. Використовуючи інструмент "Текст", додайте назву компанії або бренду поруч із графічним елементом.
4. Налаштуйте колір заливки (Fill) та колір контуру (Stroke) для всіх елементів. Спробуйте застосувати лінійний градієнт.
5. Згрупуйте всі елементи об'єкта разом.
6. Експортуйте готовий логотип у формат `.svg`.
